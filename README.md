# Cells - Deferred Rendering with WebGPU

This is a simple renderer to learn WebGPU in Rust (via wgpu-rs) and try out a couple of techniques:

- Deferred Rendering
- Instanced meshes
- Screenspace Ambient Occlusion
- Lambert Lighting
